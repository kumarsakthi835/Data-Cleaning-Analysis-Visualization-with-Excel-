# 📊 E-Commerce Sales Analysis using Microsoft Excel

## 📌 Project Overview

This project analyzes an E-Commerce Sales dataset using Microsoft Excel. It demonstrates data cleaning, table relationships, Excel formulas, PivotTables, descriptive statistics, and an interactive dashboard to generate business insights.

---

# 📂 Dataset

The workbook contains four worksheets:

- 📄 Sales_Fact – Sales transactions
- 👥 Customer_Dim – Customer details
- 📦 Product_Dim – Product information
- 🏪 Store_Dim – Store information

---

# 🎯 Assignment Tasks Completed

- ✅ Organized data using Excel Tables.
- ✅ Cleaned data by removing duplicates, handling missing values, correcting formatting, and validating data.
- ✅ Removed irrelevant columns (where applicable).
- ✅ Applied Excel formulas for calculations and analysis.
- ✅ Combined tables using XLOOKUP/VLOOKUP.
- ✅ Created PivotTables for data summarization.
- ✅ Performed descriptive statistics (Mean, Median, Mode, Standard Deviation, Min, Max, Variance, etc.).
- ✅ Built PivotCharts and an interactive dashboard.
